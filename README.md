# 📱 Contact Book App

A simple Contact Book application built with Python using Functions and Dictionaries.

## ✨ Features

* ➕ Add Contact
* 👀 View Contacts
* 🔍 Search Contact
* 🗑️ Delete Contact
* 🚪 Exit Application

## 📚 Concepts Used

* Python Functions (`def`)
* Dictionaries (`{}`)
* While Loop
* Conditional Statements (`if-elif-else`)
* User Input
* Dictionary Operations

## 🎮 How It Works

1. User selects an option from the menu.
2. Contacts are stored in a dictionary.
3. Users can add, view, search, and delete contacts.
4. The application keeps running until Exit is selected.

## 🛠 Functions Used

* `add_contact()`
* `view_contact()`
* `search_contact()`
* `delete_contact()`
